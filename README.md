# 🏏 IPL Data Analysis (2008–2024)

## 📌 Project Overview

This project focuses on **exploratory data analysis (EDA)** of the **Indian Premier League (IPL)** dataset from **2008 to 2024**.
The analysis is performed using **Python data science libraries** to uncover key insights about teams, players, and match trends.

---

## 🎯 Objectives

* Analyze **1000+ IPL matches** and **200,000+ ball-by-ball deliveries**
* Find **top performing players and teams**
* Understand **run trends, win patterns, and venue impact**
* Visualize important insights using graphs

---

## 🛠️ Tools & Technologies

* **Python**
* **Pandas** – Data Cleaning & Manipulation
* **NumPy** – Numerical Operations
* **Matplotlib** – Data Visualization

---

## 📊 Key Analysis Performed

✔ Total matches played each season
✔ Top run scorers in IPL history
✔ Most successful IPL teams
✔ Toss impact on match results
✔ Runs distribution per over
✔ Venue wise performance

---

## ⭐ Key Finding

👉 **Virat Kohli is the all-time top scorer in IPL with 8014 runs.**
